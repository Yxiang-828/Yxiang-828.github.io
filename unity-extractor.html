Extract texture assets from Unity game files and convert them to PNG format. Perfect for developers, artists, and researchers.

Key Features

  • Direct extraction from Unity3D bundles
  • PNG conversion for all Texture2D and Sprite assets
  • Batch processing with recursive directory support
  • CLI interface for automation
  • Organized output by source structure

Usage

# Single file
py unity_image_extractor.py "file.unity3d"

# Entire folder (recursive)
py unity_image_extractor.py "directory" -r

# Custom output
py unity_image_extractor.py "source" -o "output_dir" -r
