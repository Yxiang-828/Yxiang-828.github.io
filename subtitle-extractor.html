Extract subtitles from videos, or if none exist, extract MP3 audio. Built with FFmpeg for reliable media processing.

Key Features

  • Checks for embedded subtitle tracks
  • Extracts SRT subtitles if available
  • Falls back to MP3 audio extraction
  • Always produces output - never fails empty
  • Simple batch file interface

Usage

extract_cc.bat "path\to\video.mp4"

Output:
- extracted_audio/ (always created)
- extracted_captions/ (if subtitles found)
